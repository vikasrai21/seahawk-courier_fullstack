🦅 Sea Hawk Tracking Widget

Embed real-time shipment tracking on any website with a single line of code.

Live Demo

How to embed on your website

<!-- 1. Add a container div where you want the widget -->
<div id="seahawk-tracker"></div>

<!-- 2. Add this script tag -->
<script
  src="https://seahawk-courierfullstack-production.up.railway.app/embed/tracker.js"
  data-container="seahawk-tracker"
  data-theme="light"
></script>

Options

data-theme        "light" or "dark"
data-brand-color  Custom hex color, e.g. "#1a56db"
data-awb          Pre-fill an AWB number
data-container    ID of your container div

Dark theme example

<div id="tracker-dark"></div>
<script
  src=".../embed/tracker.js"
  data-container="tracker-dark"
  data-theme="dark"
  data-brand-color="#3b82f6"
></script>
