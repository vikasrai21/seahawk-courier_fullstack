# Delhivery API Setup

## How to Get Your API Key (FREE)

### Step 1 — Register as a Delhivery Partner
1. Go to **https://app.delhivery.com**
2. Click Sign Up → Register as **Courier Partner / Seller**
3. Enter your details:
   - Business Name: Sea Hawk Courier & Cargo
   - GSTIN: 06AJDPR0914N2Z1
   - Address: Shop 6 & 7, Rao Lal Singh Market, Sector-18, Gurugram – 122015
4. Wait 2–3 business days for approval email

### Step 2 — Generate API Token
1. Log in to app.delhivery.com
2. Click **Settings → Developer → API Access**
3. Click **Generate Token**
4. Copy the token (it looks like: `a1b2c3d4e5f6...`)

### Step 3 — Add to Railway Environment Variables
In your Railway dashboard → your backend service → **Variables**:

```
DELHIVERY_API_KEY        = paste_your_token_here
DELHIVERY_API_URL        = https://track.delhivery.com
DELHIVERY_WAREHOUSE      = Primary
DELHIVERY_SELLER_NAME    = Sea Hawk Courier & Cargo
DELHIVERY_SELLER_ADDRESS = Shop 6 & 7, Rao Lal Singh Market, Sector-18
DELHIVERY_SELLER_CITY    = Gurugram
DELHIVERY_SELLER_STATE   = Haryana
DELHIVERY_SELLER_PIN     = 122015
```

### Step 4 — Verify It's Working
Once deployed, open your portal and go to:
`https://your-api.up.railway.app/api/delhivery/status`

You should see: `"configured": true`

## What the API Enables

| Feature | Without Key | With Key |
|---|---|---|
| Live tracking | Opens Delhivery website | Real scan events in portal |
| AWB generation | Manual entry | Auto-generated by Delhivery |
| Shipping label | Manual | Download PDF directly from portal |
| PIN check | Always shows serviceable | Real serviceability data |
| Cancel shipment | Manual call | One click in portal |

---

## API Endpoints Available

```
GET  /api/delhivery/status              — check if key is configured
GET  /api/delhivery/track/:awb          — live tracking events
POST /api/delhivery/book                — create shipment, get AWB
GET  /api/delhivery/label/:awb          — download shipping label PDF
POST /api/delhivery/cancel/:awb         — cancel before pickup
GET  /api/delhivery/serviceability?pin= — check if PIN is serviceable
```

## Note on Sandbox vs Production
- Delhivery provides a **sandbox environment** for testing: `https://staging-express.delhivery.com`
- Use `DELHIVERY_API_URL=https://staging-express.delhivery.com` while testing
- Switch to `https://track.delhivery.com` for production
